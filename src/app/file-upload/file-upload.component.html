Pagos Realizados

folder

Es para subir el archivo que nos envia la segunda con todos los pagos realizados.
Tiene que ser ser CSV separado por comas.
Para estar seguros, lo abren con Excel, le dan guardar como y seleccionan el tipo csv delimitado por comas. El archivo tiene que tener en la primer fila el nombre de las columnas.

Las columnas que si o si tiene que tener son:

PRODUCTO POLIZA SUPLE CUOTA EXPEDIENTE EMISION MONEDA COBRADO VENCIMIENTO FECCOBRO HORCOBRO NROLOTE NROTERM MOVIMIENTO COMISION PRIMA PREMIO

Actualiza Cuotas lo que hace sincronizar las cuotas que estan en la 13017,13097 y 13120 con la base de datos de esta app.

Listado de Operaciones

folder

El listado de operaciones es un reporte que se obtiene desde la segunda.
Contiene todos lo movimientos que se hacen en expedientes relacionados con la cooperativa.
Se sube tal cual se descarga.
La primer fila contiene los encabezados.

Las columnas que si o si tiene que tener son:

'Referencia','Secc','Supl/End','CdPer','Premio','Estado_Poliza'
